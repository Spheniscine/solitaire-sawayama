# Sawayama Solitaire

A web-and-mobile-friendly implementation of Sawayama Solitaire.

Note that drag-and-drop is not supported or needed. Moves are done by selecting the card or stack, then clicking its destination.

Play here: (ad-supported) [https://www.onlinemathlearning.com/sawayama-solitaire-game.html]

Or here: [https://spheniscine.github.io/solitaire-sawayama/]<br>
